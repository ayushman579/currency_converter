## Setup Instructions
1. Clone the repo:
```bash
git clone https://github.com/your-username/currency-tracker-app.git
cd currency-tracker-app
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
python main.py
```

4. Run unit tests:
```bash
python -m unittest discover tests
```
